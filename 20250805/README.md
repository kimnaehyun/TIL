# 디버깅

1. 중단점(breaking point) 찍기
   : 함수호출하는 부분, 내가 세세히 알고싶은 부분

2. Debug 'main' : 단축키 F5

3. step into는 함수 안으로 들어가기 : F11
   step over는 다음단계 진행 : F10

4. 중간에 정지하고싶으면 빨간색 네모버튼 누르면 된다.
